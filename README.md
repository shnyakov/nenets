# Nenets Language Processing

A collection of Jupyter notebooks for Nenets language processing: creating a word frequency dictionary, training GPT-2 on Nenets, developing a simple verb lemmatization algorithm, and building a neural machine translation model using Keras.

## üìÇ Repository Structure
- `notebooks/frequency_dictionary.ipynb` ‚Äî Parsing data from various sources in the Forest Nenets language and creating a word frequency dictionary
- `notebooks/Implementation of GPT-2 & Training Language Model in Nenets Language.ipynb` ‚Äî Fine-tuning GPT-2 on Nenets language
- `notebooks/lemmatization_algorithm.ipynb` ‚Äî Developing a simple verb lemmatization algorithm for Nenets
- `notebooks/parsing_nvinder.ipynb` ‚Äî Parsing data from the *Nyaryana Vynder* newspaper
- `notebooks/multilingual_dataset_extension.ipynb` ‚Äî Multilingual dataset extension for English, French, Finnish, and Hungarian languages
- `notebooks/neural_translation_model.ipynb` ‚Äî Neural machine translation model using Keras
- `notebooks/data/input/` ‚Äî Raw datasets
- `notebooks/data/output/` ‚Äî Processed datasets
- `requirements.txt` ‚Äî Dependencies
